## 🧬 Self-Evolving Engine

**Автономный цикл каждые 30 минут:**

```
┌─────────┐     ┌─────────┐     ┌─────────┐     ┌─────────┐
│ OBSERVE │ ──► │  THINK  │ ──► │   ACT   │ ──► │  LEARN  │
│         │     │         │     │         │     │         │
│ metrics │     │ Gemini  │     │ sandbox │     │ save    │
│ errors  │     │ analyze │     │ test    │     │ results │
│ patterns│     │ propose │     │ deploy  │     │ skills  │
└─────────┘     └─────────┘     └─────────┘     └─────────┘
     ▲                                               │
     └───────────────── REPEAT ──────────────────────┘
```

### Фазы:

1. **OBSERVE** - Собирает метрики системы, ошибки из логов, паттерны из Learning Collector
2. **THINK** - Gemini AI анализирует и предлагает одно маленькое улучшение
3. **ACT** - Тестирует код в изолированном sandbox, деплоит только если успешно
4. **LEARN** - Записывает результат в Self-Reflection (успех или причину неудачи)

## 💡 Как это работает

```
Ты спишь...
    │
    ▼
💓 Brain бьёт "сердце" → проверяет все системы
    │
    ▼
🔗 Sync синхронизирует LEFT ↔ RIGHT brain
    │
    ▼
🧬 Evolution запускает цикл:
    │
    ├─► OBSERVE: метрики, ошибки, паттерны
    ├─► THINK: Gemini предлагает улучшение
    ├─► ACT: sandbox тест → deploy если OK
    └─► LEARN: записывает результат
    │
    ▼
📚 Learning Collector находит новые паттерны
    │
    ▼
🪞 Reflection сохраняет уроки
    │
    ▼
Ты просыпаешься → Система ЛУЧШЕ чем была! 🚀
```
